## Project features: 
- Model-View approach that separates data from visuals
- Event-driven interaction framework using reflection that attaches game logic to data
- Math involving position calculation on a circle
- Uses DOTween for good-looking animations wired through code 
- Camera movement
- Modular, decoupled systems
- Composite planets - a planet can have a different shape, color or face
